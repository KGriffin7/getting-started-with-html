Kyren's Notes How The Web Works

The web is a collection of computer the communicate with each other to relay inormation. The computer uses HTTP to ask servers for a particular webpage or HTML Document.

HTML

Html is described a the "the heart of the web." Any and all webpages would not exist if not for HTML. HTML itself is made of many individual aspects to keep it together. The actual text content is what is seen and read on the webpage. The markup is how it is displayed on the webpage.


HTML Markup

HTML markup is control by elements known as tags and attributes.Tags are used to alter the appearance of specific words or phrases. All tags are used in the middle of "<" and ">" To make a word bold, you would use the "b" tag and to italize, the "em" tag would be used.Attributes are complementary assets to some tags. An anchor tag, for example, would be accompanied by the attribute "href", while "src" or search would be used for an image or "img" tag.


Span and Div

Span and Div are two classes that are used to contain a specific word or phrase. Div class is a block element, meaning it creates a "box" around the specified word or phrase. Span is an inline element, similar to "a","br","strong",etc.
